## À propos
**PCS Plomberie** est une entreprise spécialisée dans les services de plomberie, de chauffage, de climatisation et d'installations sanitaires. Notre site web permet aux clients de découvrir nos services, de consulter nos tarifs indicatifs, de demander un devis, et de nous contacter facilement.

### Services offerts :
- **Piscine** : Travaux d'installation et d'entretien des systèmes de plomberie de piscines.
- **Chauffage** : Pose et réparation de systèmes de chauffage, chaudières, radiateurs, etc.
- **Sanitaire** : Installation de systèmes sanitaires (toilettes, lavabos, douches, etc.).
- **Climatisation** : Installation professionnelle de systèmes de climatisation pour les particuliers et entreprises. *(Note : pas de services de réparation pour la climatisation)*.

Notre équipe de plombiers professionnels est à votre service pour garantir un travail de haute qualité, que ce soit pour des dépannages d'urgence ou des installations complètes. Nous proposons également des **devis gratuits** sur demande, ajustés en fonction de la complexité des travaux.

### 1. **Présentation des services**
Le site présente de manière claire et concise les différents services proposés par l'entreprise, allant des installations sanitaires aux systèmes de chauffage et climatisation.

Une section dédiée aux tarifs est incluse, permettant aux clients d'avoir une idée générale des coûts pour différents types d'interventions.

## Structure des pages

### Page d'accueil
- **Sections** : Accueil, Services, Tarifs, Devis, Contact
- **Description** : La page d'accueil fournit une vue d'ensemble des services de PCS Plomberie, incluant une brève description de chaque service avec des liens pour en savoir plus.

## Comment installer et utiliser le projet

### Prérequis
- Serveur Web (Apache, Nginx)
- PHP (version recommandée : 7.4 ou supérieure)
- Base de données (MySQL ou autre)

### Installation
1. **Cloner le dépôt** :
   ```bash
   git clone https://github.com/votre-utilisateur/pcs-plomberie.git
   ```
2. **Configurer la base de données** :
   - Importez le fichier SQL fourni dans votre base de données.
   - Modifiez le fichier de configuration pour ajouter les informations de connexion à la base de données.
   
3. **Lancer le site** sur un serveur local ou distant.

### Utilisation
- Accédez à la page d'accueil pour consulter les services.
- Les clients peuvent demander un devis via la page dédiée.
- Utilisez la page contact pour toute demande spécifique.
